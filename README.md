<h2 align="center">
  Para testar
</h2>

```bash
  #  clone o repositório
  git clone link

  #  navegue até o repositório clonado
  cd memory-game

  #  baixe as dependências
  yarn

  #  Para ios:
    - cd ios
    - pod install
    - cd ..
    - yarn ios
   
  #  Para android:
    - yarn android
```
